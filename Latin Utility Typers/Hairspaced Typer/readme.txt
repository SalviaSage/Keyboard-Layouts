Hairspaced Typer;

A normal keyboard layout that types punctuation marks with a very narrow space between them.
It can improve the legibility of texts and give a better aesthetic look.

It affects the following keys:
Parenthesis
Square Brackets
Curly Brackets
Hyphen
Equals Sign
Semicolon
Colon
Comma
Period
Middle Dot
Ellipsis
Exclamation Mark
Question Mark
Number Sign
Dollar Sign
