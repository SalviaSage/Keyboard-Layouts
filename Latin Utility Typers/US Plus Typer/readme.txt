A feature of this keyboard is that it lets you press the Function keys F13 to F24 by pressing Ctrl+Shift+F# keys.

Ctrl+Shift+F2 = F14

	This keyboard features the Kana lock modifier key.
This is done by pressing the Ctrl+Pause/Break key on the keyboard.

	This keyboard layout features the AltGr(AltGraph) modifier key.
This is done by pressing the right alt key on the keyboard.

We can type superscript numbers and letters by turning on the Kana lock(Ctrl+Pause key): 0123456789 | ⁰¹²³⁴⁵⁶⁷⁸⁹
	abcdefghijklmnopqrstuvwxyz | ᵃᵇᶜᵈᵉᶠᵍʰᶦʲᵏˡᵐⁿᵒᵖ𐞥ʳˢᵗᵘᵛʷˣʸᶻ
We can type superscript capital letters by turning on the Kana lock + shift (incomplete set, missing x,y,z):
	ABCDEFGHIJKLMNOPQRSTUVWXYZ | ᴬᴮꟲᴰᴱꟳᴳᴴᴵᴶᴷᴸᴹᴺᴼᴾꟴᴿ꟱ᵀᵁⱽᵂ
Just remember to exit superscript typing by pressing Ctrl+Break again.

From the numpad, we can type subscript numbers by pressing AltGr+NumpadDigit: 0123456789 | ₀₁₂₃₄₅₆₇₈₉

Accent marks (diacritics) used in European languages can be typed by pressing AltGr and top row number keys `-3.
Non-breaking hyphen can be typed by pressing AltGr+hyphen-minus character next to the top row number keys.
Some extra European letters and variations can be typed with the keys by pressing AltGr.

(…) Ellipsis, or three dots can be typed as a single character by pressing AltGr+FullStop.
(·) Middle Dot can be typed by pressing AltGr+comma.
(–) En Dash can be typed by pressing AltGr+ -.
(—) Em Dash can be typed by pressing Shift+AltGr+ -.
(¤) Currency sign can be typed by pressing Shift+AltGr+9.

(“) Opening quotation mark can be typed by pressing AltGr+".
(”) Closing quotation mark can be typed by pressing AltGr+'.
(‘) Opening quotation mark can be typed by pressing Shift+AltGr;".
(’) Closing quotation mark can be typed by pressing Shift+AltGr+'.

(«) Opening guillemets can be typed by pressing Shift+Control+;.
(») Closing guillemets can be typed by pressing Shift+Control+'.

	Copyright symbols can be typed by pressing Ctrl+Shift+Letter keys, such as:

Ctrl+Shift+w: Circled Wz / Warenzeichen grapheme
Ctrl+Shift+r: Registered Sign
Ctrl+Shift+t: Trade Mark Sign
Ctrl+Shift+p: Sound Recording Copyright
Ctrl+Shift+c: Copyright Sign

	The numpad also has some extra features placed there:

AltGr+NumpadMinus types the prime symbol.
AltGr+ShiftNumpadMinus types the double prime symbol.

AltGr+NumpadDot types the degree sign.
Ctrlr+NumpadDot types the comma for thousands separation.

(÷) "Division Sign" can be typed with Shift+num/.
(⁄) "Fraction Slash" can be typed with Ctrl+num/ or AltGr+num/ and helps type fractions appropriately.

(×) "Multiplication Sign" can be typed with Shift+num*.
(⋅) "Dot Operator" can be typed with ctrlt+num* which is another way to write multiplication.

(−) "Minus Sign" can be typed with Shift+num-.
(=) "Equals Sign" can be typed with Shift+num+.

(≤) "Less Than or Equal To" can be typed with Ctrl+Shift+,
(≥) "Greater Than or Equal To" can be typed with Ctrl+Shift+.
	or
(≤) "Less Than or Equal To" can be typed with Ctrl+Shift+num-.
(≥) "Greater Than or Equal To" can be typed with Ctrl+Shift+num+.

	Other Features:

Ctrl+\backslash types the double vertical line (‖).
Shift+ctrl+\backslash types the broken bar (¦).
AltGr+v: Letter ç / Ç

The sleep/standby key can be pressed by pressing ctrl+shift+pause key.
