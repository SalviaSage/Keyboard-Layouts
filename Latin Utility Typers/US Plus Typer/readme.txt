US PLUS TYPER;

The "US Plus Typer" is the US keyboard, but with:
	• Function keys from F13 to F24.
	• Superscript letters and numerals.
	• Subscript letters and numerals.
	• Extra orthographic symbols.
	• Extra letters.
	• Diacritics.
	• Extra space characters.
	• Currency signs.
	• Simple mathematical support.

	• A feature of this keyboard is that it lets you press the Function keys F13 to F24 by pressing Ctrl + Shift + F# keys:
(F13)  Ctrl + Shift + F1
(F14)  Ctrl + Shift + F2
(F15)  Ctrl + Shift + F3
(F16)  Ctrl + Shift + F4
(F17)  Ctrl + Shift + F5
(F18)  Ctrl + Shift + F6
(F19)  Ctrl + Shift + F7
(F20)  Ctrl + Shift + F8
(F21)  Ctrl + Shift + F9
(F22)  Ctrl + Shift + F10
(F23)  Ctrl + Shift + F11
(F24)  Ctrl + Shift + F12

	• Extra orthographic symbols.
(¡) Shift + Ctrl + 1 "inverted exclamation mark"
(℆) Shift + Ctrl + 2 "cada una"
(№) Shift + Ctrl + 3 "numero sign"
(¢) Shift + Ctrl + 4 "cent sign"
(‰) Shift + Ctrl + 5 "per mille sign"
(☆) Shift + Ctrl + 6 "white star"
(†) AltGr + 6 "dagger"
(‡) AltGr + Shift + 6 "double dagger"
(⁊) Shift + Ctrl + 7 "tironian sign et"
(⹋) AltGr + 7 "triple dagger"
(⸸) AltGr + Shift + 7 "turned dagger"
(⚹) Shift + Ctrl + 8 "sextile"
(⁂) AltGr + 8 "asterism"
(※) AltGr + Shift + 8 "reference mark"
(⸺) Shift + Ctrl + 9 "two-em dash"
(⸻) Shift + Ctrl + 0 "three-em dash"
(§) AltGr + 9 "section sign"
(¤) AltGr + Shift + 9 "currency sign"
(¶) AltGr + 0 "paragraph sign/pilcrow"
(⸿) AltGr + Shift + 0 "capitulum"
(‑) Control + Shift + (-) "non-breaking hyphen"
(–) AltGr + (-) "en dash"
(—) AltGr + Shift + (-) "em dash"
(≠) Control + Shift + (=) "not equal to"
(≈) AltGr + (=) "approximately equal to"
(≡) AltGr + Shift + (=) "identical to"

	• This keyboard features the Kana lock toggle key.
This is done by pressing the Ctrl + Pause/Break on the keyboard.

	• This keyboard layout features the AltGr(AltGraph) modifier key.
This is done by pressing the right alt key on the keyboard.

• We can type the superscript numerals and letters by turning on the Kana Lock(Ctrl + Pause):
	0123456789 | ⁰¹²³⁴⁵⁶⁷⁸⁹
	abcdefghijklmnopqrstuvwxyz | ᵃᵇᶜᵈᵉᶠᵍʰᶦʲᵏˡᵐⁿᵒᵖ𐞥ʳˢᵗᵘᵛʷˣʸᶻ

• We can type superscript capital letters by turning on the Kana Lock + Shift(incomplete set, missing x, y, z):
	ABCDEFGHIJKLMNOPQRSTUVWXYZ | ᴬᴮꟲᴰᴱꟳᴳᴴᴵᴶᴷᴸᴹᴺᴼᴾꟴᴿ꟱ᵀᵁⱽᵂ

• We can type the subscript numerals by pressing AltGr + Numpad Number Key:
	0123456789 | ₀₁₂₃₄₅₆₇₈₉

• We can type the subscript numerals and letters by turning on the Kana Lock(Ctrl + Pause) and then pressing Ctrl + Key:
	0123456789 | ₀₁₂₃₄₅₆₇₈₉
	abcdefghijklmnopqrstuvwxyz | ₐ ₑ ₕ ᵢ ⱼ ₖ ₗ ₘ ₙ ₒ ₚ ᵣ ₛ ₜ ᵤ ᵥ ₓ (incomplete set, missing b, c, d, f, g, q, w, y, z)

Just remember to exit superscript/subscript typing by pressing Ctrl + Pause/Break again.

• Accent marks(diacritics) used in European languages can be typed by pressing AltGr and top row number keys (`)-1-2-3.
Non-breaking hyphen can be typed by pressing AltGr + hyphen-minus character next to the top row number keys.
Some extra European letters and variations can be typed with the keys by pressing AltGr + Letter key.

(·) "middle dot" can be typed by pressing AltGr + (,).
(…) "three dots" (ellipsis) can be typed as a single character by pressing AltGr + (.)

(–) "en dash" can be typed by pressing AltGr + (-).
(—) "em dash" can be typed by pressing Shift + AltGr + (-).

(«) "opening guillemets" can be typed by pressing Shift + Control + (;).
(») "closing guillemets" can be typed by pressing Shift + Control + (').
(“) "opening quotation mark" can be typed by pressing AltGr + (").
(”) "closing quotation mark" can be typed by pressing AltGr + (').
(‘) "opening single quotation mark" can be typed by pressing Shift + AltGr + (;).
(’) "closing single quotation mark" can be typed by pressing Shift + AltGr + (').

	• Copyright symbols can be typed by pressing Ctrl + Shift + Letter, such as:
(©) Ctrl + Shift + C: Copyright Sign
(🅪) Ctrl + Shift + M: Raised MC sign (French copyright sign)
(℗) Ctrl + Shift + P: Sound Recording Copyright
(®) Ctrl + Shift + R: Registered Sign
(™) Ctrl + Shift + T: Trade Mark Sign
(🄮) Ctrl + Shift + W: Circled Wz / Warenzeichen grapheme (German copyright sign)

	• Currency symbols can be typed by pressing AltGr + Letter Key, such as:
($) Shift + 4: Dollar Sign
(¢) Shift + Ctrl + 4: Cent Sign
(€) AltGr + 4: Euro Sign
(£) AltGr + Shift + 4: Pound Sign
(₿) AltGr + B: Bitcoin Sign
(¥) AltGr + Y: Yen Sign

	• Extra Spaces:
Ctrl + Space: Hairspace
Ctrl + Shift + Space: Zero-Width-No-Break-Space
AltGr + Space: Three-Per-Em-Space

	• The numpad has some extra features placed there that has to do with numerals:
(∶) "ratio" can be typed with Ctrl + NumPad(.).
	(4∶3) (12∶00 AM)

(′) "prime" can be typed with Shift + AltGr + NumPad(-).
(″) "double prime" can be typed with AltGr + Shift + NumPad(-).
	(5′) (5′ 7″)

(°) "degree sign" can be typed with AltGr + NumPad(.).
	(360°)

(÷) "division sign" can be typed with Shift + NumPad(/).
(⁄) "fraction slash" can be typed with Ctrl + NumPad(/).

(×) "multiplication sign" can be typed with Shift + NumPad(*).
(⋅) "dot operator" can be typed with ctrlt + NumPad(*), which is another way to write multiplication.

(−) "minus sign" can be typed with Shift + NumPad(-).
(=) "equals sign" can be typed with Shift + NumPad(+).

(≤) "less than or equal to" can be typed with Ctrl + Shift + NumPad(-).
(≥) "greater than or equal to" can be typed with Ctrl + Shift + NumPad(+).
	or
(≤) "less than or equal to" can be typed with Ctrl + Shift + (,)
(≥) "greater than or equal to" can be typed with Ctrl + Shift + (.)

(±) "plus-minus sign" can be typed with AltGr + NumPad(+).
(∓) "minus-or-plus sign" can be typed with AltGr + Shift + NumPad(+).

	• Other Features:

AltGr + Shift \ types the rightwards arrow (→).
