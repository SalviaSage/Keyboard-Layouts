Phoenician Typer;
https://en.wikipedia.org/wiki/Phoenician_alphabet

It has no case and is very basic.

Shift+Spacebar is the Phoenician word separator which looks like a middle dot.

Top row 1-6 has number letters.

Writing / Typing direction: Right to Left

To switch the direction of writing to Left-To-Right, insert the LTO control character by hitting the 8 key.
The 7 key will make it type Left-To-Right again.
