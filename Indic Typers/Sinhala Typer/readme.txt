Sinhala Typer;

Prenasalised consonants can be typed by pressing AltGr + Consonant Key.

Archaic numerals can be typed at the numpad by pressing Ctrl + Numpad Key.
There are more archaic numerals located at the top row number keys, typed by pressing ctrl + shift + number key.

A conjunct "R" that comes right after the consonant letter can be typed by pressing the shift+R key.
"Ref" or "Rephaar" can be typed by pressing ctrl+shift+R key.

Western Style digits can be typed by pressing AltGr + Numpad key.
