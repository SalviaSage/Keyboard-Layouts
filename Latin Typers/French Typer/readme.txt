French Typer;

Types the French language and the Portuguese language.

The diacritics can be typed by holding right alt and pressing the top row 1-2-3-4 keys.

Extra letters typed by pressing AltGr + Letter:
AltGr + Q = À
AltGr + W = È
AltGr + E = Ê
AltGr + R = É
AltGr + T = Ë
AltGr + Y = Ÿ
AltGr + U = Û
AltGr + I = Î
AltGr + O = Ô
AltGr + A = Â
AltGr + H = Ù
AltGr + J = Ü
AltGr + K = Ï
AltGr + C = Ç
AltGr + N = Ñ
AltGr + M = 🅪
AltGr + Shift + M = 🅫
