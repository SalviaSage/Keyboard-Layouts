Chinese Pinyin Typer;

Fully types Pinyin and some extra diacritics.
Switch between the Pinyin Latin alphabet and the English Latin alphabet by pressing the scroll lock key.

Type the tonal diacritics by first typing the vowel and pressing the top row number keys 1 to 4.

Chinese (Church Romanization) Typer;

Fully types Pinyin and some extra diacritics.
Switch between the Pinyin Latin alphabet and the English Latin alphabet by pressing the scroll lock key.

Type the tonal diacritics by first typing the vowel and pressing the top row number keys 1 to 4.

This is the same thing as the pinyin typer but has a few additional letters and diacritics on the top rows.

Chinese Pinyin Typer (Fullwidth);
This is the fullwidth version of the same keyboard layout.
Except, all of the letter keys A to Z are not fullwidth, but all the other characters on the keyboard are.
This program is intended to be used with a Chinese Hanzi IME program for typing Chinese characters.

The way it works is that it will read the word written in Pinyin with the correct tone diacritics.
Then the typist presses (Tab) to get a menu from where the Chinese character can be typed.
