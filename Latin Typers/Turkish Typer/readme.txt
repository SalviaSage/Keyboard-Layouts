Turkish Typer;
Also types the Kurdish Hawar alphabet and the Azerbaijani alphabet.

You can type the vowel letters with circumflex accents by pressing AltGr + Vowel letter.
You can add the circumflex accent on top of vowel letters with the backslash key (\).

The alternative version uses macrons instead of circumflex accents to indicate long vowels.

Types:
	Turkish Latin Alphabet
	Azerbaijani Latin Alphabet
	Turkmen Latin Alphabet (with RightAlt + Letter)
	Tatar Latin Alphabet
	Common Turkic Alphabet 2024 (with RightAlt+Letter)
		Transliteration of Old Turkic Script
		Kurdish Hawar Latin Alphabet
