Serbian Latin Typer:
Types in the Gaj's Latin alphabet.


Serbian Latin-Cyrillic Typer:
Types in the Gaj's Latin alphabet and if you press scroll lock, it will type in Serbian Cyrillic.
Press scroll lock again to go back to typing in Latin.
For Cyrillic, the caps lock does not work and the capital letters are typed by pressing shift + letter.

Alternatively, can install the Serbian Latin and Serbian Cyrillic typer separately.
